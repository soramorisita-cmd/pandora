#### Weidian ✅ 成功
- requests ではページが JS レンダリング前のため価格なし
- **Playwright + networkidle 待機** で `¥価格` パターンが取得可能
- 正規表現: `¥\s*([\d]+(?:\.[\d]+)?)`
- テスト結果: 380 CNY 正常取得

#### Jadeship ✅ 成功
- Playwright で同様のパターンで取得可能
- テスト結果: 178 CNY 正常取得

#### Taobao ❌ ブロック
- 直接アクセス: ログイン壁 / 重度の bot 対策でブロック
- モバイル URL (`h5.m.taobao.com`) も同様
- Jadeship/Taobao URL (`jadeship.com/item/taobao/ID`) も価格なし
- **現在の対応: スキップ（1,412件は価格 null のまま）**
- 将来的な対策候補: サードパーティ API、ブラウザプロファイル使用

## 3. フロントエンドについて（変更不要と判明）

`catalog.html` の `extractJPY()` 関数はすでに `price_jpy` を優先参照している：

```javascript
function extractJPY(p){
  const j = p.price_jpy;
  if(j && !['undefined','null',''].includes(String(j).trim())){
    return '¥' + parseInt(String(j).replace(/[^0-9]/g,'')).toLocaleString()
  }
  // フォールバック: タイトルから正規表現で抽出（精度約60%）
}
```

→ **products.json に price_jpy が入れば自動で正しい価格が表示される。HTMLの変更は不要。**
