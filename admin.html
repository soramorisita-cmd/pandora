🔧 使い方

  1. 「dataフォルダを開く」をクリックし、pandora/data フォルダを選択
  2. 全ブランドのJSONが一括読み込みされます
  3. ブランドフィルター・カテゴリフィルター・検索で絞り込んで編集
  4. 「💾 上書き保存」→ 変更があったブランドのJSONファイルのみ直接上書きされます
  5. ターミナルで git add data/ && git commit -m "fix types" && git push
