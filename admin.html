🔧 使い方

  1. 上のエリアに products.json をドラッグ&ドロップ
  2. 「タイプ編集」: ドロップダウンで正しいカテゴリを選択
  3. 「カテゴリ移動」: カードをドラッグして別のカテゴリ列にドロップ
  4. 「保存してダウンロード」→ ダウンロードされた products.json を pandora フォルダに上書き
  5. ターミナルで git add products.json && git commit -m "fix types" && git push
